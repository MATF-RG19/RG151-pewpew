# RG151-pewpew
PewPew

## _OPIS_
Igra u kojoj je cilj stići što dalje. Aviončić možete pomerati levo desno kako bi izbegli prepreke a povremeno ispaljuje i metkiće koji vam dodatno pomažu.

## _KOMANDE_
| Dugme         | Akcija           |
| :-----------: |:---------------: |
| S           | Startovanje igre     |
| D           | Skretanje desno        |
| A           | Skretanje levo |
|R            | Ponovno pokretanje igre |
| ESC           | Izlazak iz igre  |
